# ambru.biz

A modern, responsive one-page website for **ambru.biz**, a business service brand focused on practical AI consulting, AI marketing systems, company automation, and AI learning for entrepreneurs.

## Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui-inspired component patterns
- lucide-react icons
- GitHub Pages deployment workflow

## Sections

- Sticky responsive navbar
- Premium split hero with AI business dashboard visual
- Value bar
- Problem and pain point cards
- Services cards
- Business impact and AI system diagram
- Process timeline
- Practical use cases
- AI learning modules
- Credibility/testimonial placeholders
- Conversion CTA
- Contact form
- Footer

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The repository includes `.github/workflows/deploy.yml`, which builds the Vite project and deploys `dist` to GitHub Pages. A `public/CNAME` file is included for `ambru.biz`.
